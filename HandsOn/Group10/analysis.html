3. Estrategia de nombrado

A continuación, se detallan una serie de consideraciones seguidas para alcanzar una estrategia de nombrado. Y, posteriormente, se detalla una tabla a modo de ejemplo.

  • El dominio elegido posee un nombre relacionado con los datasets elegidos, y sobre el contenido que se va a desarrollar.
  • Se usan "Hash URIs (#)" para términos ontológicos, y "Slash URIs (/)" para aquellos individuales.
  • Para palabras separadas por espacios, se va a insertar un guión "-", como se muestra en el dominio plasmado en la tabla.
