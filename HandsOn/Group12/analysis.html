    Rutas de autobús

    • Description: Rutas de autobuses relacionadas con Alcobendas
    • CSV Name File: routesautobus.csv
    • Data Link: Data
    • License Link: License
    Fields of the set Value Range Info
    route_id Integer>=1 Identifica una ruta de forma exclusiva.
    agency_id String Identifica una empresa para la ruta especificada.
    route_short_name String Nombre corto de una ruta
    route_long_name String Nombre completo de una ruta
    route_type 0-7
  • 0: tranvía, metro ligero. Cualquier metro ligero o sistema ferroviario en superficie dentro de un áreametropolitana.
  • 1: subterráneo, metro. Cualquier sistema ferroviario subterráneo dentro de un área metropolitana.
  • 2: tren. Utilizado para viajes de larga distancia.
  • 3: autobús. Utilizado para rutas en autobús de corta y larga distancia.
  • 4: transbordador, ferry. Utilizado para servicio de transporte público fluvial o marítimo de corta y larga distancia.
  • 5: funicular. Utilizado para funiculares en superficie en donde el cable pasa por debajo del vehículo.
  • 6: cabina, vehículo suspendido de un cable. Normalmente se utiliza para medios de transporte público por cable en donde el vehículo queda suspendido de un cable.
  • 7: funicular. Cualquier sistema diseñado para recorridos con una gran inclinación.
    		•
    route_url Formato URL URL de una página web relativa a una ruta concreta.
    route_color Hexadecimal (ej. 00FFFF)
  • Color correspondiente a una ruta.
  • Si no se especifica ningún color, el color de ruta predeterminado es el blanco (FFFFFF)
    		•
    route_text_color Hexadecimal
  • Color legible para el texto incluido sobre un fondo del valor route_color.
  • Si no se especifica ningún tipo de ubicación, el color del texto predeterminado es el negro (000000)
    		•

    Servicios de autobús

    • Description: Servicios de las rutas de autobuses relacionadas con Alcobendas
    • CSV Name File: tripsautobus.csv
    • Data Link: Data
    • License Link: License
    Fields of the set Value Range Info
    route_id Integer>=1 Identifica una ruta de forma exclusiva.
    service_id Integer>=1 Identifica de forma exclusiva un conjunto de fechas en el que el servicio se encuentra disponible en una o más rutas.
    trip_id Integer>=1 Identifica un viaje.
    trip_headsign String Texto que aparece en un cartel que identifica el destino del viaje para los pasajeros.
    trip_short_name String Texto que aparece en horarios y carteles para que los pasajeros identifiquen el viaje.
    direction_id valor binario
  • 0 : viaje en una dirección (p. ej., viaje que sale)
  • 1: viaje en la dirección opuesta (p. ej., viaje que llega).
    		•
    wheelchair_accessible 0-2
  • 0 (o vacío): indica que no hay información de accesibilidad para el viaje.
  • 1: indica que en el vehículo utilizado para este viaje en particular hay accesibilidad para al menos un pasajero con silla de ruedas.
  • 2: indica que no hay accesibilidad para pasajeros con silla de ruedas en este viaje.
    		•

    Paradas de autobús

    • Description: Datos de las paradas de autobuses relacionadas con Alcobendas
    • CSV Name File: stopsautobus.csv
    • Data Link: Data
    • License Link: License
    Fields of the set Value Range Info
    stop_id String Identifica de forma exclusiva a una parada o estación.
    stop_code String Texto corto o un número que identifica de forma exclusiva laparada de los pasajeros.
    stop_name String Nombre de una parada o estación.
    stop_desc String Descripción de una parada.
    stop_lat Debe ser una latitud WGS 84 válida Latitud de una parada o estación.
    stop_lon Debe ser una latitud WGS 84 válida entre -180 y 180. Longitud de una parada o estación.
    zone_id A,B1,B2,B3 Zona tarifaria de un ID de parada.
    stop_url URL URL de una página web relativa a una parada concreta.
    location_type Valor binario
  • Identifica si este ID de parada representa a una parada o estación.
  • 0 o en blanco: parada. Ubicación en la que los pasajeros suben a bordo o bajan de un medio de transporte público.
  • 1: estación. Estructura física o área que contiene una o más paradas.
    		•
    parent_station String Paradas que se encuentran físicamente en el interior de estaciones.
    stop_timezone Europe/Madrid Zona horaria en la que se ubica esta parada o estación.
    wheelchair_boarding Identifica si es posible acceder en silla de rueda a la parada o estación especificada.
  • 0 (o vacío): indica que no hay información de accesibilidad para la parada.
  • 1: indica que un pasajero en silla de ruedas puede subir a algunos vehículos en esta parada.
  • 2: no es posible acceder en silla de rueda a esta parada.
    		•
