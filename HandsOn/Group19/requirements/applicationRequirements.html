Application requirements

Nuestra aplicación tomará el histórico de datos referente a la contaminación acústica en la ciudad de Madrid.

  • Al usuario se le presentará un mapa del ruido para las zonas estudiadas haciendo uso del histórico de datos
  • El usuario podrá elegir ver datos filtrando por zona
  • Podrá también ver estadísticas de la progresión y cambio en los valores para un cierto periodo de tiempo escogido
  • Además, podrá solicitar una estimación de valores de contaminación acústica para una zona concreta basándose en un número de meses anteriores escogido
