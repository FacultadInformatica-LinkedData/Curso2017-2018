Dataset requirements

  • R1. Data are in the smart cities domain.
    • Si
  • R2. Data are available as a CSV file.
    • Todos los datasets que usamos menos uno están disponibles en formato CSV. La información sobre los puntos de recarga eléctrica está disponible en ficheros excel (extensión XLS).
  • R3. Data have an open license so it can be published.
    • Debido a que todos los datos provienen del ayuntamiento de madrid, todos se encuentran bajo las mismas condiciones de uso que se pueden ver en: http://datos.madrid.es/portal/site/egob/menuitem.400a817358ce98c34e937436a8a409a0/?vgnextoid=b4c412b9ace9f310VgnVCM100000171f5a0aRCRD
  • R4. Data can be easily linked with generic real-world entities (e.g., locations)
    • Si. Al tratarse de la localización de plazas de parking es posible relacionar esta información con entidades como: Localidades, Sitios de interés, etc.
  • R5. Documentation exists for the data (optional).
    • Si, se puede encontrar en: http://datos.madrid.es/FWProjects/egob/contenidos/datasets/ficheros/Estructura_conjunto_datos_www%20madrid%20es_v7.pdf
  • R6.  Data can come from multiple data sources (optional).
