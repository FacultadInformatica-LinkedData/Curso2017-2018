CENTROS DE EDUCACIÓN

Columna Tipo Comentarios/Rango
NOMBRE xsd: string Nombre del centro de educación. Unique, not empty values.
HORARIO xsd:string Indica el horario de apertura del centro. Not unique, empty values. Range(00:00-23:59, Lunes-Domingo).
SERVICIO_RUTA xsd:boolean Indica si el centro dispone de servicio de ruta o no. Not unique, not empty values. Range(TRUE/FALSE).
COMEDOR xsd:boolean Indica si el centro dispone de comedor o no. Not unique, not empty values. Range(TRUE/FALSE).
TRANSPORTE xsd:string Indica el tipo de transporte que se puede utilizar para llegar al centro. Not Unique, not empty value.
DESCRIPCIÓN xsd:string Pequeña descripción del centro que indica el tipo de centro y las actividades que imparte. Unique, empty values.
ACCESIBILIDAD xsd:boolean Indica si el centro está adaptado para gente con discapacidaz física. Not unique, not empty values. Range(TRUE/FALSE).
CONTENT-URL xsd:anyURI Indica si el centro está adaptado para gente con discapacidaz física. Not unique, not empty values.
CLASE-VIA xsd:string Indica el tipo de via. Not unique, not empty values.
NOMBRE-VIA xsd:string Indica el nombre de la via. Not unique, not empty values.
TIPO-NUM xsd:string Indica si es una autopista o el numero de la calle. Not unique, not empty values.
NUM xsd:int Indica el número de la calle donde se encuentra el centro. Not unique, not empty values. Range(0-9999).
ORIENTACIÓN xsd:string Indica una información extra en cuanto a la ubicación de centro para saber hacia donde mira el edificio. Not unique, empty values.
CODIGO-POSTAL xsd:int Indica el código postal al que corresponde el centro. Not unique, empty values. Range(28000-28999).
BARRIO xsd:string Indica el barrio al que pertenece el centro. Not unique, not empty values.
DISTRITO xsd:string Indica el distrito al que pertenece el centro. Not unique, not empty values.
LATITUD xsd:float Indica una de las coordenadas del centro. Unique, not empty values.
LONGITUD xsd:float Indica una de las coordenadas del centro. Unique, not empty values.
TELÉFONO xsd:string Indica el teléfono del centro. Unique, not empty values. Range(00000000-99999999).
FAX xsd:string Indica el FAX del centro. Unique, empty values. Range(00000000-99999999).
EMAIL xsd:string Indica el email del centro. Unique, empty values.
TIPO xsd:anyURI Indica el tipo de centro educativo. Not unique, not empty values.
